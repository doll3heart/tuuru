# Tuuru 验收作品

这里有两套内容完全离线、可重复生成的生产导入样例；JSON 和 PNG 各自包含同一份作品数据。

## 最短验收路线

1. 打开 `http://127.0.0.1:8765/reader/`，分别导入两个 `.json` 或 `.png` 文件。
2. 文章《雾港第七码》的密码是 `2468`。填写主角称呼后，至少走两条路线：
   - 站台 → 论坛 → 夜市 → 天台 → 结局
   - 联系人 → 档案室 → 夜市 → 档案记录 → 结局
3. 在文章各节点逐个打开七类手机卡片：消息、相册、联系人、论坛、备忘、购物、浏览。
4. 在《雾港口袋终端》按桌面位置打开七个 App。消息里检查聊天、动态、通话和可重选回复；论坛里检查帖子、嵌套回复和选项；相册检查相册二级页；购物检查购物车/订单页签。
5. 备忘、相册、浏览、购物首次进入会显示作者指定的“林弦设备”接入确认，这是预期行为。
6. 两部作品都带全屏交叉斜排水印；文章是文字水印“纯代乙向禁止偷吃 · Tuuru 验收样例”，纯手机是图片印章水印。

重新生成：`node scripts/generate-acceptance-works.mjs`

注意：纯手机里的壁纸、桌面顺序和复杂皮肤用于验证作品结构及初始呈现；读者端仍保留最终的本地美化与阅读习惯控制权。
